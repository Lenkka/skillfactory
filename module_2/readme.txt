Описание проекта:

I. Основные цели и задачи проекта

Суть учебного проекта — отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, находящихся в группе риска.

Целью проекта было привести данные в пригодный для дальнейшего исследования вид и построить файл-основу для дальнейшего использования: предсказания результатов госэкзамена по математике для каждого ученика.

Задачи проекта: очистить данные и провести их преобразование и оценку.

Учебные задачи проекта: применить на практике изученное в пройденных модулях курса.

II. Краткая информация о данных

Данные были предоставлены в достаточно полном виде с минимумом ошибок.
Это сведения о школьниках, сдававших экзамен по математике. Они учитывают практически все условия жизни и обучения школьников в общем виде. К анализу пригодны после минимального вмешательства.

III. Этапы работы над проектом

1. Выполнить вспомогательное задание и сделать выводы
2. Составить план анализа на основе вспомогательного задания
3. Просмотреть и оценить данные
4. Заполнить пропуски и исправить неверные значения
5. Отобрать данные, которые влияют на конечный результат - оценку ученика на экзамене
